Pokédex

¿Qué es la Pokédex?

La Pokédex es una enciclopedia virtual que contiene información de todas las especies de Pokémon conocidas. Puedes usar la API para obtener datos detallados sobre cada Pokémon.

API de la Pokédex

La API de la Pokédex te permite acceder a información detallada sobre los Pokémon, incluyendo sus habilidades, estadísticas, y mucho más. Visita la PokéAPI para más detalles.
